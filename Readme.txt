This project used Junit4 and hamcrest library for unit testing.
Application is written as plain java program where build and project dependencies are managed by Maven.
Main program is CalculatorImpl class which read data from console to process it.
This uses Shunting Yard Algorithm for processing inline operator mathematical expression to Reverse polish notation and process it in a simpler way.
This assumes all operations performed are left associated. 

This assumes no need for a logger to log as of now and used sys out to print on console
